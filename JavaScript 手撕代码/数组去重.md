## 数组去重的方法

数组去重的方法，在没有查阅其他资料的情况下，我脑子里想到最直接的方法是使用 `Set` 实现，其次是 `for` 循环，比如

`Set` 实现

```js
const arr = [1, 2, 3, 5, 5, 2, 4];

function unique(arr) {
  return [...new Set(arr)];
}

console.log(unique(arr)); //[1, 2, 3, 5, 4]
```

`for` 循环实现

```js
const arr = [1, 2, 3, 5, 5, 2, 4];

function unique(arr) {
  const result = [];
  for (let i of arr) {
    /**
     * 逻辑是检查 result 中是否已有相同元素，不存在才添加
     * 使用 result.indexOf(i) === -1 || !result.includes(i) 都可以
     */
    if (!result.includes(i)) {
      result.push(i);
    }
  }
  return result;
}

console.log(unique(arr)); //[1, 2, 3, 5, 4]
```

用 `Array` 的实例方法实现：`filter` + `indexOf`

```js
const arr = [1, 2, 3, 5, 5, 2, 4];

function unique(arr) {
  return arr.filter((item, index) => arr.indexOf(item) === index);
}

console.log(unique(arr)); //[1, 2, 3, 5, 4]
```

利用 `Map` 也能实现

```js
const arr = [1, 2, 3, 5, 5, 2, 4];

function unique(arr) {
  const result = [];
  const map = new Map();
  for (let i of arr) {
    if (!map.has(i)) {
      map.set(i, true);
      result.push(i);
    }
  }
  return result;
}

console.log(unique(arr)); //[1, 2, 3, 5, 4]
```

## 总结

纵观上面这几种方法，实现数组去重的方法大致就两类：
- 利用 `Set` 结构的唯一性
- 循环遍历数组来检查是否唯一
